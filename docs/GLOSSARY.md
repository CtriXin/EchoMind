# Glossary

- **Decision Record**: A ledger entry that captures the decision context, available options, user choice, reason, result, and preference signal evidence.
- **Pattern Memory**: A distilled memory of user-prefers and user-dislikes signals with confidence, counts, and evidence from Decision Records.
- **xmem card**: The rule-shaped export record EchoMind promotes from qualifying Pattern Memory signals into xmem recall.
