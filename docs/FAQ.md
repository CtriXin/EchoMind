# FAQ

## What is EchoMind?

EchoMind is a decision ledger + pattern distiller. It records why choices were
made, what happened afterward, and which preference signals are worth carrying
forward.

## What is EchoMind not?

EchoMind is not a workflow runner. It does not execute automations, change MMS
routes, or rewrite runtime routing behavior.

## What commands should I start with?

Use `echomind help` to see the CLI entry points.

Use `npm run smoke` to run the repository smoke check.
